# Changelog

All notable changes to the `core-demographics-lk` pack will be documented in this file.

## [1.0.0] - 2024-01-19

### Added
- Initial release
- NIC number variables (combined, old format, new format)
- Basic patient demographics (name, DOB, gender)
- Sri Lanka administrative divisions:
  - Province (9 provinces)
  - District (25 districts)
  - DS Division (Divisional Secretariat)
  - GN Division (Grama Niladhari)
- Contact information (address, mobile, home phone, email)
- Validation patterns for Sri Lankan formats:
  - NIC: Old (9 digits + V/X) and New (12 digits)
  - Mobile phone: 07X XXXXXXX
  - Landline: 0XX XXXXXXX
